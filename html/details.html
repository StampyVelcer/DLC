Description:

This game was developed for Diseño de Lenguajes de Consulta course (Query Language Design, or DLC) of the fourth of Ing. en Sistemas de Información.


    Game features:
  • Software Rendering (Doesn't use Direct3D nor OpenGL, the rendering and rastering is performed by Java).
  • Several rasterization strategies (Wireframe, Solid, Affine Texture Mapping y Perspective Correct Texture Mapping).
  • AI developed with FSM (Finite State Machine).
  • Map generated through a bitmap, considering each RGB channel (R: Roof, G: Wall, B: Floor).

    Tools used:
  • Netbeans 6.9
  • MS Paint
